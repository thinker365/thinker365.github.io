# 命令模式

  1. ×
    • 删除光标所在的字符
  2. nd
    • n代表数字，表示剪切光标所在位置的后面的个字符，注意：以空格结束，不要按回车键
  3. dd
    • 剪切光标所在行的内容，用当于1dd
  4. ndd
    • n代表数字，表示剪切光标所在行的下面n行内容
  5. yy
    • 复制光标所在行的内容
  6. nyy
    • n代表数字，表示复制光标所在行的下面n行内容
  7. p
    • 小写字母，粘贴[在光标下一行粘贴出来]
  8. P
    • 大写字母，粘贴[在光标上一行粘贴出来]
  9. u
    • 撤销上一步编辑操作
  10. gg
    • 光标跳转到第一行
  11. GG
    • 光标跳转到最后一行

# 终端模式

在这种模式下，开发者可以通过在绵辑器的最后一行输入vm提供的指令完成一系列的缩粗器的控制操作，或完成文本的搜索，替换和控制等操作，甚至可以在该模式下，使用linux提供的命令完成来完成其他的命令调用。 进入末行模式，可以在命令模式状态下，使用快捷键冒号:进入。

  1. 显示或隐藏文件行号
    • set nu
    • set nonu
  2. 跳转到指定行
    • :行号
  3. 查找内容
    • :/内容
    • :?内容
    • 此时，可以通过键盘中的n（往下）或N(往上)查找匹配项
  4. 退出
    • :q，退出编辑器，如果没有办法退出，则强制退出
    • :q!，强制退出
  5. 保存
    • :w，保存内容
    • :w文件名，把当前内容另存为指定文件名
    • :wq，先保存内容，接着退出vi编辑器
  6. 文本替换
    • :%s/原内容/新内容/，把原内容替换为新内容，替换一次
    • :%s/原内容/新内容/g，把原内容全部替换为新内容
    • :%s/原内容//，把原内容匹配到的删除
    • :%s/原内容//g，把原内容全部删除
  7. 粘贴保持格式
    • :set paste，再进入编辑模式操作粘贴
